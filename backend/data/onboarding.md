# Employee Onboarding Guide

## Day 1

### Morning (9:00 AM - 12:00 PM)
- Meet with HR for paperwork and badge issuance
- Receive laptop and equipment setup
- Set up email and accounts
- Review security policy and sign NDA

### Afternoon (1:00 PM - 5:00 PM)
- Meet with manager for team introduction
- Set up development environment
- Review codebase documentation
- Pair programming session with mentor

## Week 1

- Complete all required training modules
- Set up 1:1 meetings with team members
- Review product roadmap and current sprint
- Begin working on first ticket under supervision
- Attend all team standups and ceremonies

## Week 2

- Take ownership of first independent task
- Participate in code review process
- Present work in sprint demo
- Complete 30-day check-in with manager
- Set up goals for first 90 days

## First 90 Days

### Goals
- Complete onboarding checklist
- Ship first feature to production
- Contribute to team documentation
- Build relationships with cross-functional partners
- Understand company culture and values

### Check-ins
- Day 30: Performance check-in with manager
- Day 60: Mid-point review and goal adjustment
- Day 90: Full performance review

## Benefits Enrollment

Benefits must be enrolled within 30 days of start date. Visit the HR portal to select:
- Health insurance plan
- Dental and vision coverage
- 401(k) contribution percentage
- Life insurance coverage

## Equipment

Standard equipment includes:
- Laptop (MacBook Pro or Dell XPS)
- 27-inch monitor
- Keyboard and mouse
- Headset
- Request additional equipment through IT with manager approval
